1.Property และ Value คืออะไรแตกต่างกันอย่างไร

Property คือ ชื่อเรียกของ element นั้นๆ ส่วน Value คือค่าของตัว element น้ั้นๆ เช่น Color (Color คือ Property) = Red (red คือ Value)

2.Property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

Property ที่สอนใน precamp มี 5 กลุ่ม

 
        1. Property for content
        2. Property for border
        3. Property for padding
        4. Property for margin
        5. Property for element

3.Value unit มีกี่ประเภทอะไรบ้าง แต่ละประเภทคืออะไร

Value unit มี 2 ประเภท

Absolute lenghts Relative lenghts
Cm, mm, in, px, pt, pc em, ex, ch, rem, vh, vw, vmin, vmax, %

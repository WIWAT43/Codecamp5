2.ให้หาความเหมือนและความแตกต่างระหว่าง 2 Tag ที่กำหนดให้

Attribute Differentiation
<hr> & <br> <hr> tag to define a thematic change in the content
<br> tag inserts a single line break
<ul> & <ol> <ul> tag defines an unordered (bulleted) list.
<ol> tag defines an ordered list. An ordered list can be numerical or alphabetical
<select> & <checkbox>& <radio button> <select> defines a drop-down list
<checkbox> defines a checkbox
<radio button> defines a radio button
<div> & <span> <div> defines a division or a section in an HTML document
<span> tag is used to group inline-elements in a document
<script> & <style> <script> define a client-side script
<style> define style information for an HTML document
